# Getting Started with PostgreSQL

Welcome to the PostgreSQL tutorial! This section introduces you to PostgreSQL, the world's most advanced open-source relational database system.

## Contents

1. [What is PostgreSQL?](what-is-postgresql.md) - Learn about PostgreSQL, its history, key features, and why it's a popular choice for modern applications.

## About This Tutorial

This tutorial is designed to help you learn PostgreSQL from the ground up. Whether you're new to databases or coming from another database system, you'll find comprehensive guides and examples to get you started.

### Who This Tutorial Is For

- Developers new to PostgreSQL
- Database administrators learning PostgreSQL
- Students and educators
- Anyone interested in modern database systems

### What You'll Learn

- PostgreSQL fundamentals
- SQL query language
- Database design principles
- Advanced PostgreSQL features
- Best practices for performance and security
- Working with serverless PostgreSQL (Neon)

## Prerequisites

- Basic understanding of databases (helpful but not required)
- Familiarity with command-line interfaces
- A text editor or IDE for writing SQL queries

## Getting Help

If you need help or have questions:

- Check the [official PostgreSQL documentation](https://www.postgresql.org/docs/)
- Visit [Neon Documentation](https://neon.com/docs) for serverless-specific features
- Join the [PostgreSQL community](https://www.postgresql.org/community/)
- Search [Stack Overflow](https://stackoverflow.com/questions/tagged/postgresql) for common issues
